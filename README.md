## Requirements

* PHP >= 5.3

## Setup

```sh
$ git clone https://github.com/chernyshevAV/Brain-Games.git

$ cd brain-games

$ make install
```

## Run brain-games
Choose any game you would like to play and enter one of the commands:
```sh
$ make brain-even

$ make brain-calc

$ make brain-progression

$ make brain-gcd

$ make brain-prime
```
